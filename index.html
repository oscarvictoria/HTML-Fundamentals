The word "tags" are interchangeable with "elements"

Some things that I learned while reading on this subject are that the main structure of a website has to have tags also know as elements to indicate what kind of format to display it on the front end.

Attributes tell us more about elements

Some word that have special meaning can be highlighted by using the bold attribute

The two properties of attributes are name and value. Each attribute must have these two. You can use attributes on most elements.
